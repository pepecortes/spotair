On the given model, replace all references to the source by the destination. Remove source if succesful
Parameters:
Name Type Description
Model Object sequelize model where the references will be replaced
refModel Object sequelize model holding the reference
refField string name of the field holding the reference to referenceModel
sourceid number reference id to replace and remove
destinationid number new reference id
Returns:
An object like {updated: 10, idRemoved: "aerodrome"} or null if failed

Object
