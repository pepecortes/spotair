Présentation du nouveau site
Pour quoi ?
Modernisation de la technologie (infrastructure) :
    • flexibilité à l’heure d’ajouter de nouvelles fonctions (ex : utilisateurs, liens avec réseaux sociaux...)
    • portabilité (pour l’hébergement ou pour le développement)
Pas de modifications graphiques significatives (mais peut-être c’est une bonne occasion ?)
Actuel :
Serveur web Apache maintenu par OVH
Serveur database MySql (OVH)
Langages de programmation : PHP + JAVASCRIPT
Nouveau :
Encapsulation DOCKER (permet la migration de hébergement et le travail en local)
NODE.JS (JAVASCRIPT) pour tous les serveurs
VUEJS (JAVASCRIPT) pour le frontend
Database toujours MYSQL
